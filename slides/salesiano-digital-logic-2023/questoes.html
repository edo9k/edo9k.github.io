Questões de Lógica e Introdução a Programação

As questões abaixo são as mesmas que vimos em sala.

O que é uma variável em JavaScript?

Uma variável em JavaScript é um nome simbólico usado para armazenar dados, como números, strings ou objetos.

Qual é a diferença entre 'let', 'const' e 'var' ao declarar variáveis em JavaScript?

'let' e 'const' são usados para declarar variáveis com escopo de bloco, enquanto 'var' possui escopo de função.

O que é um loop 'while' em JavaScript e como ele funciona?

Um loop 'while' em JavaScript executa um bloco de código repetidamente enquanto uma condição especificada for verdadeira.

Como você define uma função em JavaScript?

Uma função em JavaScript é definida usando a palavra-chave 'function' seguida pelo nome da função e parâmetros, se houver.

Qual é a saída do seguinte código JavaScript: 'console.log(5 + "5")'?

A saída será a string '55' porque ocorre uma concatenação de strings.

O que é um array em JavaScript e como você declara um?

Um array em JavaScript é uma estrutura de dados que armazena uma coleção de valores. Você pode declará-lo usando colchetes, por exemplo, 'let meuArray = [1, 2, 3]'.

Explique o que é o 'this' em JavaScript e como ele é usado.

O 'this' em JavaScript se refere ao objeto atual e é usado para acessar propriedades e métodos desse objeto.

Como você escreve um comentário de uma única linha em JavaScript?

Um comentário de uma única linha em JavaScript é precedido por '//' como em '// Este é um comentário de uma linha'.

O que é uma condicional 'if-else' e como ela é usada?

Uma condicional 'if-else' é usada para tomar decisões em JavaScript. Ela executa um bloco de código se uma condição for verdadeira e outro bloco se a condição for falsa.

Qual é a função do método 'console.log()' em JavaScript?

O método 'console.log()' é usado para imprimir mensagens ou valores no console do navegador ou no ambiente de desenvolvimento.

O que representa o operador lógico 'NÃO' (NOT) na lógica proposicional?

O operador lógico 'NÃO' (NOT) inverte o valor de uma proposição. Se a proposição for verdadeira, 'NÃO' a torna falsa e vice-versa.

Em lógica proposicional, o que é uma implicação (condicional) representada por 'Se... então...'?

Uma implicação (condicional) é verdadeira, a menos que a primeira proposição seja verdadeira e a segunda seja falsa.

Qual é a tabela verdade do operador lógico 'E' (AND)?

A tabela verdade do operador lógico 'E' (AND) só é verdadeira quando ambas as proposições são verdadeiras, caso contrário, é falsa.

O que é uma contrapositiva em lógica proposicional?

A contrapositiva de uma implicação inverte e nega as proposições originais, mantendo a mesma validade lógica.

Explique o conceito de 'Bicondicional' (SE e SOMENTE SE) na lógica proposicional.

O 'Bicondicional' (SE e SOMENTE SE) é verdadeiro quando ambas as proposições são iguais, ou seja, ambas verdadeiras ou ambas falsas.

Qual é a tabela verdade do operador lógico 'OU' (OR)?

A tabela verdade do operador lógico 'OU' (OR) é verdadeira quando pelo menos uma das proposições é verdadeira.

O que é uma tautologia em lógica proposicional?

Uma tautologia é uma proposição que é sempre verdadeira, independentemente dos valores das proposições individuais.

Como se representa uma proposição composta em lógica proposicional?

Uma proposição composta é representada pela combinação de proposições individuais usando operadores lógicos.

O que é a regra de simplificação em lógica proposicional?

A regra de simplificação permite simplificar proposições complexas removendo cláusulas redundantes.

Como se representa uma negação em lógica proposicional?

A negação de uma proposição é representada pelo operador 'NÃO' (NOT).
